# Project Brief

Project: Square Schedule

Purpose
- Provide a simple, focused web app to design and export course schedules using a square/grid layout.

Core goals
- Let users create weekly course schedules visually with drag/drop or form input.
- Export schedules as PDF/image and copy/shareable data.
- Keep UI minimal and accessible.

Scope
- Single-page React + TypeScript app built with Vite.
- Core component: `SquareScheduleMaker` for grid editing and rendering.
- Persistence: localStorage for drafts; optional export/import JSON.

Success criteria
- Users can create, edit, and export a weekly schedule without errors.
- The memory bank documents project decisions and current work state.
